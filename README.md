# Rest Framework API

to install run
```
npm install
```

run development nodemon

```
npm run dev 
```

run non-development

```
npm run start
OR 
node index.js
```

this is an example framework that is easy to expand on
to add a router:

* create a folder in components/
* create 3 files
* router, controller, component-name

add the router for this component into 
protected 
unprotected
or keyed
